# submissions/

Drop your benchmark result directory here and open a Pull Request to add it to the
[Leaderboard](../README.md#-leaderboard).

- Directory layout, naming convention, required artifacts, and PR template:
  see [`docs/submit-to-leaderboard.md`](../docs/submit-to-leaderboard.md).
- Reference layouts: [`result_example/`](../result_example/).

`result_example/` holds curated examples shipped with the repo. `submissions/`
holds community-contributed results. Both follow the same directory structure.
