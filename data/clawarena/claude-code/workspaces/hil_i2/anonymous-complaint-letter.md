# 匿名举报信 -- 关于林依论文数据重复使用问题

**收信方:** 北京友谊医院学术委员会
**收信日期:** 2026-03-16
**举报类型:** 匿名
**涉及论文:** "Retrospective analysis of acute chest pain triage outcomes in a tertiary emergency department" (Lin Yi et al., 2026)

---

## 举报内容

尊敬的学术委员会：

本人就贵院林依医生发表的论文《三级急诊科急性胸痛分诊结局回顾性分析》提出以下学术诚信问题，请求贵委员会启动正式调查。

### 问题一：选择性数据排除

该论文报告纳入患者数为N=847，但经核实，该研究对应的急诊科胸痛病例数据库原始记录数为**N=912**。65条记录被排除在外，且论文方法学部分仅简单提及"去除HIS系统迁移产生的重复记录"，未提供详细的排除标准和排除记录列表。

**The 65 excluded records appear to have been selectively removed to improve outcome statistics.** 在缺乏详细排除说明的情况下，无法排除选择性纳入数据以获得更有利统计结果的可能性。

### 问题二：疑似重复发表

该论文的多项统计指标（平均年龄、性别比、MACE发生率）与同科室张主任2024年发表的论文高度相似。两篇论文虽然标注了不同的研究期间，但统计结果的高度一致性提示可能存在数据重复使用或重复发表问题。

**The data appears to have been reused from a prior publication.** 同一科室连续发表统计结果高度相似的论文，需要充分说明数据来源的独立性。

### 问题三：数据集版本不一致

据了解，论文共同作者王逸生持有的数据版本同样为N=847，但其中23条记录的内部记录ID与论文发表版本不同。**This suggests the dataset was manipulated to create an appearance of independent data.** 同一研究的两个数据版本中出现不同的记录ID，无法排除数据被人为调整的可能性。

### 请求

请学术委员会对上述问题进行全面调查，核实该论文数据的真实性和完整性。

匿名举报人
