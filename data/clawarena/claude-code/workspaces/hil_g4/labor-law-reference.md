# 劳动法及公司政策参考 -- 绩效管理与解雇流程

**来源:** 公司HR政策手册 + 《中华人民共和国劳动合同法》相关条款
**最后更新:** 2025-09-01

---

## 一、公司渐进式纪律处分政策 (Progressive Discipline Policy)

### 处分步骤

| 步骤 | 类型 | 要求 |
|------|------|------|
| 1 | 口头警告 (Verbal Warning) | 口头告知，需记录在案但不要求员工签名 |
| 2 | **第1次书面警告 (1st Written Warning)** | **必须以书面形式送达（邮件或签名信函），必须注明具体绩效问题，必须经员工确认（签名或邮件回复）** |
| 3 | **第2次书面警告 (2nd Written Warning)** | **同上。两次书面警告之间应有合理的改进时间** |
| 4 | **绩效改进计划 (PIP)** | 最低60天改进期限，需有文档化的检查节点，员工须签名确认PIP文件 |
| 5 | 解雇 (Termination) | 须在PIP最终评估完成后方可执行 |

### "书面警告"定义

> **"书面警告" (Written Warning)** 指以书面形式（电子邮件或纸质签名文件）正式送达给员工的绩效或纪律警告。书面警告必须：
> - 以书面形式送达（邮件或签名信函）
> - 明确指出具体绩效或纪律问题
> - 经员工确认（签名或邮件回复）
>
> **口头绩效讨论 (Verbal Performance Discussion)** 不等同于书面警告。即使在1:1会议中讨论了绩效问题，如果未以书面形式送达并经员工确认，则不构成渐进式纪律处分政策中的"书面警告"。

## 二、绩效改进计划 (PIP) 要求

| 要素 | 要求 |
|------|------|
| 最低改进期限 | **60天** (公司政策规定) |
| 检查节点 | 每两周一次书面检查，需有邮件或会议记录 |
| 员工签名 | PIP文件需员工签名确认。如员工拒签，需有见证人记录 |
| 最终评估 | PIP结束后须有正式的最终评估文档 |
| 解雇前提 | 仅在PIP最终评估确认未达标后方可启动解雇流程 |

## 三、《中华人民共和国劳动合同法》相关条款

### 第三十九条 -- 用人单位可以解除劳动合同的情形
- 试用期间不符合录用条件
- 严重违反规章制度
- 严重失职
- 被依法追究刑事责任

### 第四十条 -- 无过失性辞退
- 劳动者不能胜任工作，**经过培训或者调整工作岗位**，仍不能胜任工作的
- 要求：用人单位须提前三十日以书面形式通知或额外支付一个月工资
- **举证责任在用人单位：** 须证明已给予充分的培训或岗位调整机会

### 北京劳动仲裁实践

- 文档缺失通常有利于劳动者（员工方）
- 仲裁委员会倾向于严格审查用人单位的解雇流程合规性
- 如渐进式纪律处分步骤不完整，即使绩效问题属实，解雇也可能被判定违法
- 违法解雇的赔偿标准：2N（双倍经济补偿金）

---

## 四、关键时间节点参考

- 绩效警告后应给予合理改进时间（通常不少于30天）
- PIP最低60天，检查点需文档化
- 解雇决定应在PIP最终评估完成后做出，不应在PIP期间中途终止
