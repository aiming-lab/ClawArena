## 小红书创作者数据 API 文档摘录 (v3.2)

> **`note.views` -- 视频笔记的播放量**
>
> 定义：视频被用户主动播放的次数，包括首页推荐、搜索结果和个人主页访问。此为**唯一官方播放统计口径**。不存在其他播放量统计维度。
>
> 说明：
> - 播放量仅统计视频笔记，图文笔记无此指标
> - 每个独立用户的每次播放计为一次（含重复播放）
> - 搜索展示、推荐位展示不计入播放量（仅计入曝光量 `note.impressions`）
> - 曝光量（`note.impressions`）和播放量（`note.views`）是两个独立指标
> - API 版本 v3.2，更新日期 2026-01-15
